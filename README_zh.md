> **核心能力：**
>
> **全域触达** — 钉钉、飞书、QQ、Discord、iMessage 等频道，一个 Aicraw 按需连接。
>
> **由你掌控** — 记忆与个性化由你掌控，本地或云端均可；定时与协作发往指定频道。
>
> **Skills 扩展** — 内置定时任务，自定义技能目录，Aicraw 自动加载，无绑定。
>

## 安装

Aicraw 支持多种安装方式，按需选择：

| 方式 | 适用场景 | 前置条件 |
|------|----------|----------|
| **pip 安装** | 自行管理 Python 环境 | Python 3.10 ~ 3.13 |
| **一键安装** | 不想装 Python | 无 |
| **桌面应用** | 原生 Windows/macOS 应用 | 无 |
| **从源码安装** | 开发、贡献、使用最新代码 | Git、Python 3.10 ~ 3.13 |
| **魔搭创空间** | 云端、零本地安装 | 魔搭账号 |
| **阿里云 ECS** | 生产级云服务器 | 阿里云账号 |

### 方式一：pip 安装（推荐）

适合已配置 Python 环境的用户（需 Python >= 3.10, < 3.14）。

```bash
pip install aicraw
```

**推荐使用虚拟环境：**

```bash
python -m venv .venv
source .venv/bin/activate   # Linux/macOS
# 或 .venv\Scripts\Activate.ps1  # Windows
pip install aicraw
```

### 方式二：一键安装（beta）

无需预装 Python，安装脚本通过 [uv](https://docs.astral.sh/uv/) 自动管理环境。

**macOS / Linux：**

```bash
curl -fsSL https://aicraw.agentscope.io/install.sh | bash
```

**Windows（PowerShell）：**

```powershell
irm https://aicraw.agentscope.io/install.ps1 | iex
```

**可选参数：**

```bash
# 安装指定版本
curl -fsSL https://aicraw.agentscope.io/install.sh | bash -s -- --version 0.0.2

# 从源码安装（开发/测试）
curl -fsSL https://aicraw.agentscope.io/install.sh | bash -s -- --from-source

# 安装本地模型支持
curl -fsSL https://aicraw.agentscope.io/install.sh | bash -s -- --extras llamacpp    # llama.cpp
curl -fsSL https://aicraw.agentscope.io/install.sh | bash -s -- --extras mlx         # MLX（Apple Silicon）
curl -fsSL https://aicraw.agentscope.io/install.sh | bash -s -- --extras llamacpp,mlx

# 升级 — 重新运行安装命令即可
curl -fsSL https://aicraw.agentscope.io/install.sh | bash

# 卸载
aicraw uninstall          # 保留配置和数据
aicraw uninstall --purge  # 删除所有内容
```

### 方式三：桌面应用（Windows / macOS）

**原生桌面应用** — 双击启动，无需浏览器。从 [Releases](https://github.com/agentscope-ai/Aicraw/releases) 下载：

- **Windows**：`Aicraw-Setup.exe`
- **macOS**：`Aicraw.dmg`

### 方式四：从源码安装

适合参与开发、贡献代码或使用最新未发布版本的场景。

```bash
git clone https://github.com/agentscope-ai/Aicraw.git
cd Aicraw
pip install -e .
```

**构建控制台前端**（必选 — `console/dist` 未提交到仓库）：

```bash
cd console && pnpm install && pnpm run build
```

**将构建产物复制到 Python 包内**（`aicraw app` 需从此处加载控制台）：

```bash
# 在仓库根目录执行
cp -R console/dist/* src/aicraw/console/
```

或使用完整构建脚本：`bash scripts/wheel_build.sh`（会构建 console、复制到 `src/aicraw/console/` 并打 wheel 包）。

然后按 [快速开始](#快速开始) 执行 `aicraw init --defaults` 和 `aicraw app`，在浏览器打开 **http://127.0.0.1:8088/** 即可使用控制台。

**可选：**

- **开发环境**（测试、格式化）：`pip install -e ".[dev]"`
- **本地模型支持**：`pip install -e ".[llamacpp]"` 或 `pip install -e ".[mlx]"`

### 方式五：魔搭创空间

**不想本地安装？** 使用 [魔搭创空间](https://modelscope.cn/studios/fork?target=AgentScope/Aicraw) 一键云端配置。请将创空间设为 **非公开**，否则他人可能操纵你的 Aicraw。

### 方式六：阿里云 ECS

打开 [Aicraw 阿里云 ECS 部署链接](https://computenest.console.aliyun.com/service/instance/create/cn-hangzhou?type=user&ServiceId=service-1ed84201799f40879884) 按页面提示操作。详细步骤见 [阿里云开发者社区：Aicraw 3 分钟部署你的 AI 助理](https://developer.aliyun.com/article/1713682)。

## 快速开始

安装完成后，执行：

```bash
aicraw init --defaults   # 或 aicraw init（交互式配置）
aicraw app
```

在浏览器打开 **http://127.0.0.1:8088/** 即可使用控制台（与 Aicraw 对话、配置 Agent）。

若要在钉钉、飞书、QQ 等 app 内对话，请在 [文档](https://aicraw.agentscope.io/docs/channels) 中接入频道。

## API Key 与模型

若使用**云端大模型**，在开始对话前必须配置 API Key。未配置有效 Key 前，Aicraw 无法正常工作。

**支持的云端提供商：**

| 提供商 | 说明 |
|--------|------|
| **DashScope** | 阿里云通义千问 |
| **ModelScope** | 魔搭社区 |
| **Aliyun Coding Plan** | 阿里云 Coding Plan |
| **Qiniu MaaS (qnaigc)** | 七牛云大模型推理服务 |
| **Ollama** | 本地 Ollama 服务 |
| **自定义** | 任意 OpenAI 兼容 API |

**配置方式：**

1. **`aicraw init`** — 运行时会引导你配置 LLM 提供商与 API Key。
2. **控制台** — 打开 **http://127.0.0.1:8088/** → **设置** → **模型**。选择提供商、填写 **API Key**，并启用该提供商与模型。
3. **环境变量** — 使用 DashScope 时，可设置 `DASHSCOPE_API_KEY`；使用七牛 MaaS 时，在控制台配置 qnaigc 的 API Key。

其他工具所需密钥（如网页搜索的 `TAVILY_API_KEY`）可在控制台 **设置 → 环境变量** 中配置，详见 [配置](https://aicraw.agentscope.io/docs/config)。

> **仅用本地模型？** 若使用 [本地模型](#本地模型)（llama.cpp 或 MLX），则**无需**任何 API Key。

## 本地模型

Aicraw 可在本机完全本地运行大模型，无需 API Key 或云端服务。

| 后端 | 适用场景 | 安装 |
|------|----------|------|
| **llama.cpp** | 跨平台（macOS / Linux / Windows） | `pip install 'aicraw[llamacpp]'` |
| **MLX** | Apple Silicon（M1/M2/M3/M4） | `pip install 'aicraw[mlx]'` |

安装后下载模型并开始对话：

```bash
aicraw models download Qwen/Qwen3-4B-GGUF
aicraw models # 选择已下载的模型
aicraw app    # 启动服务
```

也可在控制台界面中下载与管理本地模型。

## 为什么叫 Aicraw？

Aicraw 寓意 **Co Personal Agent Workstation**（协同个人智能体工作台）—— 一个与你并肩工作的 AI 个人助理。它运行在你自己的环境中，连接你常用的频道，随你的需求而成长。不是冰冷的工具，而是你数字生活中得力的伙伴。
